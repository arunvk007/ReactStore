=> React component life cycle 

1.Mounting(stages of rendering dom elements in a component) => componentDidMount()
2.Update(stage of rendering after a change or updation in component data[state,props, etc]) => componentDidUpdate()
3.Unmounting(stage of removing dom elements that rendered) => componentDidUnmount()

Redux
*****
To avoid the cons of property drilling, redux is used.


            dispatch         payload             updated data
component------------>action------------>reducer-------------->store


action=> an asynchronous function defined in a component

payload=> return value or output generated by action

reducer=> It is a function that accepts an empty state and action and return a new state
